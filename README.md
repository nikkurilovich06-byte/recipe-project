# recipe-project

Проект для работы с рецептами, ингредиентами и списками покупок.

В проекте реализованы классы:

* `Ingredient` — ингредиент с названием, количеством и единицей измерения;
* `Recipe` — рецепт с названием и списком ингредиентов;
* `ShoppingList` — список покупок, который можно составлять на основе рецептов;
* `DietaryRecipe` — диетический рецепт с указанием категории, например "веган", "без глютена", "кето".

## Возможности проекта

Проект позволяет:

* создавать ингредиенты;
* объединять одинаковые ингредиенты по названию и единице измерения;
* создавать рецепты;
* добавлять ингредиенты в рецепт;
* масштабировать рецепт на нужное количество порций;
* создавать список покупок по рецептам;
* удалять рецепт из списка покупок;
* получать итоговый отсортированный список ингредиентов;
* объединять два списка покупок;
* создавать диетические рецепты.

## Установка

Склонируйте репозиторий:

```bash
git clone https://github.com/nikkurilovich06-byte/recipe-project
```

Перейдите в папку проекта:

```bash
cd recipe-project
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск тестов

Для запуска тестов используйте команду:

```bash
pytest
```

или:

```bash
python -m pytest
```
